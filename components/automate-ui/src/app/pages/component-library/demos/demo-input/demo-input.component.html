chefInput is a directive applied to the native input and textarea elements. chefInput should be wrapped in a chef-form-field.

Note: Currently the chefInput directive does very little. It's here only as a holder for now so you won't notice any differing behavior if you don't use it. However, chefInput will be necessary when more complex behaviors are implemented, so don't neglect adding it.

Angular Template Code Example

<chef-form-field>
  <label>Test Field<label>
  <input chefInput #testfield id="test" required type="text" placeholder="Type 'error' to see an example error.">
  <chef-error *ngIf="testfield.value == 'error'">This is an example error</chef-error>
</chef-form-field>
